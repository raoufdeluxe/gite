Comment héberger ce site

1. Mettez index.html et alerte.json dans le même dossier sur votre serveur.
2. Pour activer une alerte de manière permanente (même après fermeture du navigateur), modifiez alerte.json directement sur le serveur :

{
  "active": true,
  "type": "warning",
  "message": "Météo incertaine ce week-end."
}

Pour désactiver, passez active à false.
